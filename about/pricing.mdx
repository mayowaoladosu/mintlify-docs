LayerRail pricing is based on the resources you create inside a project.

## Resource-based billing

Common billed resources include:

- Virtual machines.
- GPU virtual machines.
- Managed PostgreSQL.
- Kubernetes worker nodes.
- Load balancers.
- Storage and bandwidth where applicable.
- AI inference usage.

## Start small

Use small plans for prototypes and upgrade when usage grows.

<Tip>
  For current prices, open the LayerRail console and review the create page for the resource you want to launch.
</Tip>

## Billing verification

Some projects require billing verification before infrastructure can be created.
