## Security posture

LayerRail focuses on:

- Project-scoped resources.
- SSH key access.
- API token controls.
- Private networking.
- Firewall rules.
- Audit-friendly resource state.
- Provider-backed infrastructure.

## Report a security issue

Email security-sensitive reports to:

```text
support@layerrail.com
```

Include:

- A clear description.
- Affected resource IDs if relevant.
- Steps to reproduce.
- Screenshots or logs with secrets removed.
